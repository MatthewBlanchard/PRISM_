Class Target

A 'Target' is a discriminator used to validate the target of an action.

Metamethods

target:__new(range) Creates a new Target with the specified range.

Methods

target:setRange(range, enum) Sets the range of the Target.
target:validate(owner, toValidate) Validates the target.

Methods

target:setRange(range, enum)
Sets the range of the Target.

Parameters:

  • range number The range of the Target.
  • enum string The type of range. Can be "box" or "circle".
target:validate(owner, toValidate)
Validates the target. Each Target subclass must implement this method.

Parameters:

  • owner Actor The actor performing the action.
  • toValidate table The target to validate.
