Class Actor

Represents entities in the game, including the player, enemies, and items. Actors are composed of Components that define their state and behavior. For example, an actor may have a Sight component that determines their field of vision, explored tiles, and other related aspects.
Actor.actions
A collection of the actor's innate actions. This is used mostly for actions that are 'instrinsic' to the actor, such as a spider casting a web. It might be better to use a component for this, and I may change it in the future.
  • actions table
Actor.article
The article to use when referring to the actor. Currently non-functional.
  • article string
Actor.char
Defines the actor's offset in the sprite sheet.
  • offset integer
Actor.color
Defines the actor's base color.

Fields:

  • color table
Actor.conjugate
Whether to conjugate verbs when referring to the actor. Currently non-functional.
  • conjugate boolean
Actor.emissive
Defines whether the actor's color should be used instead of it's lit color.
  • emissive boolean
Actor.name
The name of the actor.
  • name string
Actor.position
The position of the actor on the map.
  • position Vector2
Actor.pronoun
The pronoun to use when referring to the actor. Currently non-functional.
  • pronoun string
Actor.reactions
A collection of the actor's innate reactions. This is hardly ever used and marked for removal.
  • reactions table
Actor.visible
Defines whether the actor can be seen.
  • passable boolean

Class Action

A class representing an Action that an actor can take in a game. An Action consists of an owner, a name, a list of targets, and a list of target actors. This class is derived from the Object class.
Action.time
The time it takes to perform this action
Action:__new(owner[, targets])
Constructor for the Action class.

Parameters:

  • owner actor The actor that is performing the action.
  • targets table An optional list of target actors. Not all actions require targets. (optional)

Returns the number of targets associated with this action.

Returns:

    number The number of targets associated with this action.
Action:getTarget(n)
Returns the target actor at the specified index.

Parameters:

  • n number The index of the target actor to retrieve.

Returns:

    actor The target actor at the specified index.

    nil If no target actor exists at that index.
